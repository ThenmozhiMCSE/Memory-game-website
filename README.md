🧠 Memory Game Website

A fun and interactive Memory Card Game built using modern web technologies. Test your memory skills by matching pairs of cards in the least number of moves!

🚀 Features
🎮 Flip and match cards
⏱️ Timer to track your performance
🔢 Move counter
🔄 Restart / Reset game option
🎨 Responsive and attractive UI
⚡ Fast performance using Vite
